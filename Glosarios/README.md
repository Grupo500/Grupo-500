# 📚 Glosarios — Grupo 500

Colección de términos de **informática, programación e inteligencia artificial** con definiciones sencillas para estudiar.

## Cómo está organizado

Cada tema tiene su propio archivo:

| Archivo | Contenido |
|---------|-----------|
| [git-y-control-de-versiones.md](git-y-control-de-versiones.md) | Git, GitHub, repositorios (push, pull, commit…) |
| [programacion-general.md](programacion-general.md) | Términos generales de desarrollo (frontend, backend, API…) |
| [inteligencia-artificial.md](inteligencia-artificial.md) | IA, modelos, prompts, tokens… |

## Cómo usarlo

- Cada término trae: **definición sencilla**, una **analogía** para recordarlo y un **ejemplo/uso** real.
- Cuando aprendas un término nuevo, dímelo y lo agrego al archivo que corresponda.
- Para estudiar, lee un archivo a la vez y trata de explicar cada término con tus palabras.

## Formato de cada término

> ### nombre-del-término
> **Qué es:** definición corta y clara.
> **Analogía:** una comparación fácil de recordar.
> **En la práctica:** cómo/cuándo se usa.
